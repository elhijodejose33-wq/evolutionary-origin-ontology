# CONTRIBUTING.md

## Contribution Guide for the Evolutionary Origin Ontology Matrix

> **This repository does not collect opinions.**
>
> It collects structurally precise contradictions: historical, institutional, economic, technological, juridical, religious, ecological, and epistemic phenomena in which official language conceals operational inversion.

⸻

## 1. Purpose of Contributions

The Anomalies Matrix is a high-density ontology of human-system distortions. Each entry identifies a specific crossroad where an institution, discourse, regime, technology, economy, doctrine, or legal fiction presents itself as serving Life while materially administering separation, scarcity, hierarchy, captured life-energy, or false unity.

Contributors are invited to extend, correct, refine, or deepen this matrix by submitting entries that expose the architectural friction between:

- **The name a system gives itself**
- **The practice through which the system actually operates**
- **The hidden contradiction between universal rhetoric and selective enforcement**

This project is grounded in the evolutionary rule that systemic change reaches threshold through **embodied persons**. One person stops feeding the lie. Then another. One body refuses the false name. One voice stops cooperating with the official discourse. One contribution, if properly named, can weaken an entire semantic architecture.

> **Rule 9 — Embodied Persons**
>
> Systemic change reaches threshold through embodied persons. The work is not waiting for reform from above. A living person names the inversion, withdraws nourishment from false administration, and contributes to a structure that allows Life to circulate without contradiction.

⸻

## 2. What Counts as a Valid Crossroad?

A valid crossroad is not merely a topic, event, policy, ideology, technology, or institution.

A valid crossroad is a **specific systemic phenomenon** in which the official narrative and the operational mechanism diverge sharply.

The entry must reveal an **Inversion**:

- A system claims care while administering dependency.
- A system claims freedom while producing captivity.
- A system claims neutrality while encoding hierarchy.
- A system claims innovation while enclosing common inheritance.
- A system claims charity while preserving the conditions that require charity.
- A system claims truth while monopolizing interpretation.
- A system claims security while expanding surveillance.
- A system claims sustainability while relocating harm elsewhere.
- A system claims democracy while manufacturing consent.
- A system claims abundance while administering scarcity.

Do not submit generic themes such as “capitalism,” “colonialism,” “technology,” “religion,” or “education” unless the phenomenon is narrowed to a concrete structural mechanism.

Good entries are specific. Excellent entries expose the machinery.

⸻

## 3. Required Formatting Schema

Every contribution must preserve the matrix’s strict three-part sequence.

| Required Field | Function | Required Analytical Standard |
|---|---|---|
| `Phenomenon Name` | Names the exact systemic crossroad. | Must be precise, structural, and historically or institutionally recognizable. Avoid vague abstractions. |
| `The Official Discourse (The Illusion)` | States the rationalized narrative used by the status quo. | Must describe how the system justifies itself morally, legally, scientifically, economically, spiritually, or administratively. |
| `The Structural Distortion (The Practice)` | Reveals the operational contradiction beneath the narrative. | Must expose the mechanism of power, exclusion, capture, scarcity, surveillance, debt, hierarchy, extraction, or false naming. |

⸻

## 4. Perfect Submission Example

> **Phenomenon Name:** Algorithmic Tenant Screening.
>
> **The Official Discourse (The Illusion):** Tenant screening platforms claim to protect landlords, reduce fraud, improve housing-market efficiency, and generate objective risk assessments from rental, credit, and legal records.
>
> **The Structural Distortion (The Practice):** Past poverty becomes future exclusion; eviction filings, credit instability, criminalization, data errors, and neighborhood-coded disadvantage are transformed into automated housing denial, making shelter dependent on opaque statistical permission.

## 5. Ontological Quality Requirements

Every proposed entry must pass the following tests.

### 5.1 Specificity Test

The phenomenon must be narrow enough to be operationally analyzed.

| Weak Submission | Strong Submission |
|---|---|
| Technology addiction | Infinite Scroll as Interface Convenience |
| Colonialism | Mandate System Trusteeship |
| Healthcare inequality | Employer-Based Healthcare |
| Education debt | Student Debt as Investment in Self |
| Environmental hypocrisy | Carbon Offsets as Climate Neutrality |

### 5.2 False Name Test

The entry must identify the false name that stabilizes the system.

Ask:

- What does the system call itself?
- What moral vocabulary protects it?
- What institutional phrase makes the violence appear normal?
- What name would the system lose legitimacy under?

### 5.3 Practice Test

The entry must describe what materially happens.

Avoid vague condemnation. Do not write only that a system is “bad,” “unjust,” “exploitative,” or “oppressive.” Show the mechanism.

Strong mechanisms include:

- eligibility gates
- debt discipline
- legal fiction
- land enclosure
- data extraction
- algorithmic scoring
- resource monopolization
- administrative delay
- privatized infrastructure
- symbolic laundering
- credential filtering
- donor control
- surveillance feedback loops
- moralized scarcity
- contractual coercion
- institutional gatekeeping

### 5.4 Non-Redundancy Test

Before submitting, search the existing matrix. Your proposed crossroad must either:

1. Add a genuinely new phenomenon, or
2. Sharpen an existing phenomenon with a more precise mechanism, or
3. Correct a weak or historically inaccurate entry.

Do not submit duplicates with different vocabulary.

### 5.5 Ontological Alignment Test

The entry must expose at least one of the following distortions:

- **Separation:** treating persons, peoples, bodies, or ecosystems as fundamentally disposable or opposed.
- **Inversion:** using life-serving language to administer life-denying practice.
- **Captured life-energy:** converting labor, attention, land, data, care, water, knowledge, or longing into retained institutional value.
- **Administered scarcity:** creating or preserving deprivation through rules, prices, access gates, borders, credentials, debt, or ownership.
- **False unity:** imposing sameness, compliance, or centralized control while claiming reconciliation or harmony.
- **Externalized temple:** relocating authority from embodied recognition to institution, platform, state, office, credential, or doctrine.

## 7. Submission Format

Submit proposed additions as Markdown in the following structure:

```markdown
**Phenomenon Name:** [Precise, structural title.]

**The Official Discourse (The Illusion):** [The rationalized, institutional narrative of the status quo.]

**The Structural Distortion (The Practice):** [The raw, hidden mechanism of power, legibility, scarcity, extraction, or contradiction.]
```

For multiple proposed entries, repeat the structure for each phenomenon.

Do not submit entries as essays. Do not submit loose bullet points. Do not submit only titles.

⸻

## 9. Style Guide

The tone of the matrix must remain:

- academically rigorous
- historically grounded
- structurally precise
- non-sentimental
- non-partisan
- non-generic
- uncompromising toward false names
- careful about factual claims
- hostile to abstraction that hides bodies

Use dense, direct prose. Every sentence must do analytical work.

### Avoid

- slogans
- moral grandstanding
- partisan shorthand
- vague anti-system language
- mystical language without structural content
- Wikipedia-style definitions
- entries that merely describe a phenomenon without exposing the contradiction

### Prefer

- “The system calls X by the name Y, while operationally doing Z.”
- “The mechanism converts A into B under the cover of C.”
- “The official discourse claims universal principle, but enforcement is selective along lines of D.”
- “The practice turns a baseline of Life into a rent stream, risk score, eligibility gate, or administrative dependency.”

⸻

## 10. Revision Standard

Maintainers may request revisions if an entry is:

- too broad
- too vague
- historically unsupported
- rhetorically strong but mechanically weak
- duplicative
- insufficiently tied to the Illusion/Practice structure
- written as opinion rather than structural analysis
- missing the hidden mechanism of power

A rejected entry is not a rejected contributor. It is a demand for sharper naming.

⸻

## 11. Validation Before Submission

Run the validation script before opening a pull request:

```bash
python validate.py
```

The validator checks structural consistency, JSONL formatting where applicable, and required fields. It does not replace philosophical judgment. Passing validation means the entry is syntactically acceptable; it does not mean the entry has ontological force.
